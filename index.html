What is Cloud Practitioner?

The AWS Cloud Practitioner (CLF-C02) is designed for anyone who wants to demonstrate foundational AWS Cloud knowledge. Perfect for sales, marketing, project managers, and those starting their cloud journey.

Exam Details: 65 questions | 90 minutes | $100 USD | Passing score: 700/1000

Exam Domains

1. Cloud Concepts (24%)

  • Value proposition of AWS Cloud
  • Cloud economics and billing
  • Cloud architecture design principles

2. Security & Compliance (30%)

  • AWS shared responsibility model
  • AWS security services (IAM, KMS, CloudTrail)
  • Compliance programs

3. Cloud Technology & Services (34%)

  • Core AWS services (EC2, S3, RDS, VPC)
  • AWS global infrastructure
  • Deployment and operations

4. Billing, Pricing & Support (12%)

  • AWS pricing models
  • Cost management tools
  • AWS support plans
